# RSA (Rivest-Shamir-Adleman)

## Overview

RSA is a public-key cryptographic algorithm used for secure communication, encryption, and digital signatures. Unlike symmetric encryption algorithms, RSA uses a public key for encryption and a private key for decryption.

## Features

* Public-key cryptography
* Prime number validation
* Key generation
* Message encryption
* Message decryption
* Educational implementation in C++

## Files

* rsa.cpp – RSA algorithm implementation

## Purpose

This implementation was developed to demonstrate the fundamentals of asymmetric cryptography, including key generation, encryption, and decryption using the RSA algorithm.
