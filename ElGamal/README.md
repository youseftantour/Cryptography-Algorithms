# ElGamal Encryption

## Overview

This project demonstrates the ElGamal public-key cryptosystem implemented in C++. ElGamal is an asymmetric encryption algorithm based on the Diffie-Hellman key exchange principle.

The program allows users to generate keys, encrypt a message, and decrypt the ciphertext to recover the original message.

## Features

* Key generation
* Public and private keys
* Modular exponentiation
* Message encryption
* Message decryption
* Random session key generation
* User input support
* Encryption verification

## Files

* elgamal.cpp – ElGamal implementation
