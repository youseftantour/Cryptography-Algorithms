# DES (Data Encryption Standard)

## Overview

DES is a symmetric-key encryption algorithm that encrypts and decrypts data using the same secret key.

## Features

- Symmetric encryption
- 56-bit key
- Block cipher
- Educational implementation in C++

## Files

- des.cpp – DES algorithm implementation

## Purpose

This implementation was developed to understand the fundamentals of symmetric-key cryptography and block cipher operations.
