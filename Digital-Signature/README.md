# Digital Signature System

## Overview

This project demonstrates a Digital Signature System implemented in C++ using RSA cryptography principles. The program generates RSA keys, signs a message using a private key, and verifies the signature using a public key.

Digital signatures are widely used to verify authenticity, integrity, and non-repudiation in secure communications.

## Features

* Prime number validation
* RSA key generation
* Digital signature creation
* Signature verification
* User input support
* Authentication testing

* digital_signature.cpp – Digital signature implementation
