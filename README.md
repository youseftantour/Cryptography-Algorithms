# Cryptography Algorithms

This repository contains implementations of several cryptographic algorithms developed in C++.

## Algorithms

- DES
- AES
- RSA
- Diffie-Hellman
- ElGamal
- Digital Signatures
- Elliptic Curve Cryptography

## Technologies Used

- C++
- Cryptography
- Modular Arithmetic
- Information Security

## Purpose

This project was created to better understand how encryption, authentication, and key exchange algorithms work internally.
