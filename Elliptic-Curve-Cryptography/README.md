# Elliptic Curve Cryptography (ECC)

## Overview

This project demonstrates the fundamentals of Elliptic Curve Cryptography (ECC) implemented in C++. The implementation performs elliptic curve point addition, point doubling, and point negation over a finite field.

ECC is widely used in modern secure communication protocols due to its strong security and smaller key sizes compared to traditional public-key cryptography systems.

## Features

* Point representation
* Point addition
* Point doubling
* Point negation
* Modular arithmetic
* Elliptic curve operations
* User input support
* ECC operation testing
